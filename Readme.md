# 🚖 Uber Ride Booking Analytics Dashboard

A complete end-to-end **Data Analytics** project built using **Power BI** to analyze Uber ride booking data and transform raw information into meaningful business insights.

This dashboard helps monitor booking performance, revenue, ride status, vehicle performance, customer ratings, and location trends through an interactive and user-friendly interface.

---

# 📌 Project Overview

The objective of this project is to analyze Uber ride booking data and create an interactive dashboard that helps stakeholders understand business performance and make data-driven decisions.

The dashboard provides insights into:

- Total Bookings
- Completed & Cancelled Bookings
- Revenue Analysis
- Total & Average Distance
- Vehicle-wise Performance
- Monthly Booking Trends
- Pickup & Drop Location Analysis
- Customer & Driver Ratings
- Ride Status Distribution
